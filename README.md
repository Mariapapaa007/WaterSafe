Smart Water Quality Risk Checker

## Problem
Many people consume unsafe water without knowing contamination risks.

## Solution
WaterSafe provides water quality **risk awareness** based on location and water source.

## Features
- Risk level: Low / Medium / High
- Possible contaminants
- Health effects
- Safety precautions

## Technology Used
- Rule-based logic
- Simple user interface

## Note
This system provides **risk awareness**, not lab-tested water values.
